What is the recommended way to grant to an EC2 machine a read-only access to a specific file located in an S3 bucket?

Options

1. Create an IAM instance profile and use the predefined "AmazonS3ReadOnlyAccess" policy
2. Associate S3 bucket with EC2 instance
3. Adjust the security group associated with the EC2 instance
4. Add the necessary file permissions to the IAM instance profile associated with the EC2 instance
