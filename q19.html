Write a Java program that analyzes a text string and counts the occurrences of each word.
You can use any Integrated Development Environment (IDE) or online Java compiler to run this program.
You are given a string text containing lowercase words, separated by a space.
Count how many words show up in the string more than once.
If the same word shows up three times or more, your counting must be increased just by one.

Function: Implement the method countRepeatedWords
Paramters: text (string): lowercase words, each sepearted by space.
Return Value: count (int): number of repeated words.
Examples:
InputOutput
apple mango apple mango0
apple mango apple mango apple1
apple mango apple mango apple mango2
apple mango apple mango apple mango apple2
